PORTFOLIO

SUN RA TRIBUTE PAGE

I put this website together as part of a project on Free Code Camp. It was the first website I designed using bootstrap, and the goal was to demonstrate aspects I had been utilising in previous exercises. In particular, I wanted to use the responsive grid system and image thumbnails to create an article that might be found on a music blog. While creating this page, I learnt a lot about the strengths and limitations of using a framework and the importance of having a clear design brief before starting.

SMART LABELLING SITE

During my time with Smart Labeling I was with redesigning the e-commerce website to give it a more contemporary feel. I was given a design brief to keep within the companies branding, and I learnt about the importance of creating a consistent message and user experience. In addition, I also made the website responsive, significantly increasing traffic from mobile and tablet devices.

DENTAL CARDS SITE

After creating the Smart Labelling website, I was then asked to create a website for the spin-off company, 'DentalCards.co.uk'. The brief was to make to make an e-commerce site for a single product, and explain the virtues of using direct thermal print media to a new audience. During my time working on this project I learned a lot about tailoring an online experience to a specific audience.
